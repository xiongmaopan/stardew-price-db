# 🎮 StardewPriceDB 网站维护指南（小白版）

> 你不需要懂代码！只需要会复制粘贴和改数字就行。

---

## 📁 你只需要关心这一个文件

```
stardew-price-db/data/items.json
```

这个文件包含所有物品数据，改它就能更新网站。

---

## ✏️ 常见操作

### 1️⃣ 修改物品价格

打开 `data/items.json`，找到要改的物品，修改数字：

```json
{
  "name": "Parsnip",
  "basePrice": 35,      ← 改这个数字就是改售价
  "seedPrice": 20,      ← 种子价格
  ...
}
```

### 2️⃣ 添加新物品

复制一个现有物品，粘贴到末尾，修改内容：

```json
{
  "id": 100,                    ← 改成新的数字ID
  "name": "新物品名字",          ← 物品名字
  "slug": "new-item",           ← 网址用的（英文，用-连接）
  "category": "Crops",          ← 分类：Crops/Fish/Animal Products/Forage/Minerals
  "basePrice": 100,             ← 售价
  "seedPrice": 50,              ← 种子价格
  "season": ["Spring"],         ← 季节：Spring/Summer/Fall/Winter
  "growthTime": 5,              ← 生长天数
  "regrows": false,             ← 是否可重复收获 true/false
  "description": "描述文字"
}
```

### 3️⃣ 删除物品

直接删掉那一整块 `{ ... }` 就行。

---

## 🚀 改完之后怎么更新网站？

### 方法一：最简单（推荐）

1. 双击运行 `一键更新.bat`（我帮你创建好了）
2. 等它跑完，上传 `out` 文件夹到服务器

### 方法二：手动命令

在项目文件夹里打开终端，输入：
```
npm run build
```

---

## ❓ 常见问题

### Q: 改完数据网站没变？
A: 需要重新运行 `npm run build`，然后重新部署。

### Q: 改错了网站打不开？
A: 恢复 `items.json` 到之前的样子，或者找我帮忙。

### Q: 怎么知道数据格式对不对？
A: 运行 `npm run verify`，它会告诉你哪里有问题。

## 🧮 加工品价格怎么算？

不用自己算！用这个公式：

| 加工品 | 公式 |
|-------|------|
| Wine（酒） | 基础价格 × 3 |
| Juice（果汁） | 基础价格 × 2.25（向下取整） |
| Jelly（果酱） | 基础价格 × 2 + 50 |
| Pickles（腌菜） | 基础价格 × 2 + 50 |

**例子**：草莓 basePrice = 120
- Wine = 120 × 3 = 360
- Jelly = 120 × 2 + 50 = 290
